# Source-to-profile mapping

Classify curated sources by how they feed the profile. Update after inventory and rename.

**Use classes**

| Class | Meaning |
|-------|---------|
| direct profile | Ready or near-ready for CV / portfolio / public profile use |
| feedstock | Source material to distill later into derived artifacts |
| skip | Keep only if needed for archive; do not use for profile outputs |

Do not record job identifiers, Brainjob job paths, or document manifests here. Phase 2 owns Brainjob linkage.
